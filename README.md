Simple Telegram Bot (tg_bot)

Простой Telegram-бот для общения с пользователем.
Бот умеет отвечать на команды, поддерживать диалог и показывать случайные фотографии кошек через TheCatAPI.

Проект написан на Python с использованием aiogram и aiohttp.


ВОЗМОЖНОСТИ

- Приветствие пользователя
- Список доступных команд
- Ответы на простые фразы
- Получение случайного фото кошки через TheCatAPI
- Поддерживающие и дружелюбные ответы
- Асинхронная работа


КОМАНДЫ И ФРАЗЫ

Команды:
/start  – приветственное сообщение
/help   – список доступных команд и фраз

Фразы (вводятся текстом):
"Покажи кошку"   – бот пришлёт случайное фото кошки  
"Как дела?"      – бот расскажет, как у него дела  
"Поддержи меня"  – бот скажет добрые слова  
"Пока"           – прощание с ботом  

Любой другой текст:
Бот повторит сообщение и предложит продолжить общение

УСТАНОВКА И ЗАПУСК ЛОКАЛЬНО

1. Клонировать репозиторий:

   git clone https://github.com/likhaalex/tg_bot.git
   cd tg_bot

2. Установить зависимости:

   pip install -r requirements.txt

3. Создать Telegram-бота через @BotFather
   и установить переменную окружения TOKEN

   Linux / macOS:
   export TOKEN=ВАШ_TELEGRAM_BOT_TOKEN

   Windows (PowerShell):
   setx TOKEN "ВАШ_TELEGRAM_BOT_TOKEN"

4. (Опционально) Получить API-ключ на https://thecatapi.com
   и указать его в коде вместо 'CAT_API_KEY'

5. Запустить бота:

   python main.py


РАБОТА С THECATAPI

Бот использует TheCatAPI для получения случайных изображений кошек.

Используемый эндпоинт:
https://api.thecatapi.com/v1/images/search

Для стабильной работы рекомендуется:
- зарегистрироваться на сайте TheCatAPI
- указать свой API-ключ в переменной CAT_API_KEY


ДЕПЛОЙ

Бот можно развернуть на любом сервисе, поддерживающем Python:
- Railway
- Render
- VPS / сервер

Не забудьте добавить переменные окружения:
- TOKEN – токен Telegram-бота
- CAT_API_KEY – ключ TheCatAPI (по желанию)
